season

/

GET

Gibt die aktuelle Season zurück

Anfrage

GET https://api.nitroleague.de/season

Antwort

{
    "id": number,
    "name": string,
    "registerStart": Date,
    "registerEnd": Date,
    "start": Date
}

Beispiel

Anfrage

GET https://api.nitroleague.de/season

Antwort

{
    "id": 3,
    "name": "Season 12",
    "registerStart": "2022-07-03T22:00:00.000Z",
    "registerEnd": "2022-07-24T19:59:00.000Z",
    "start": "2022-07-31T22:00:00.000Z"
}

/name​

GET

Gibt die Season mit angegebem Namen zurück

Anfrage

GET https://api.nitroleague.de/season/<Name>

Antwort

{
    "id": number,
    "name": string,
    "registerStart": Date,
    "registerEnd": Date,
    "start": Date
}

Beispiel

Anfrage

GET https://api.nitroleague.de/season/Season 12

Antwort

{
    "id": 3,
    "name": "Season 12",
    "registerStart": "2022-07-03T22:00:00.000Z",
    "registerEnd": "2022-07-24T19:59:00.000Z",
    "start": "2022-07-31T22:00:00.000Z"
}

/name/group/​

GET

Gibt Informationen über eine Gruppe innerhalb einer Season zurück

Anfrage

GET https://api.nitroleague.de/season/<Name>/<Name>

Antwort

{
    "id": number,
    "name": string,
    "divisionId": number
}

Beispiel

Anfrage

GET https://api.nitroleague.de/season/Season 12/Division 1

Antwort

{
    "id": 81,
    "name": "Division 1",
    "divisionId": 18
}

/division/id/slots​

GET

Gibt Informationen über die Slots einer Division zurück (Aufstieg/Abstieg/etc.)

Anfrage

GET https://api.nitroleague.de/season/division/<ID>/slots

Antwort

[
    {
        "slot": number,
        "direction": string
    },
    {
        "slot": number,
        "direction": string
    },
    ...
]

Beispiel

Anfrage

GET https://api.nitroleague.de/season/division/18/slots

Antwort

[
    {
        "slot": 1,
        "direction": "Playoffs"
    },
    {
        "slot": 2,
        "direction": "Playoffs"
    },
    ...
]

/division-group/id/weeks​

GET

Gibt die Anzahl der Spielwochen dieser Gruppe zurück

Anfrage

GET https://api.nitroleague.de/season/division-group/<ID>/weeks

Antwort

{
    "weeks": number
}

Beispiel

Anfrage

GET https://api.nitroleague.de/season/division-group/81/weeks

Antwort

{
    "weeks": 11
}

/division-group/id/standings/week​

GET

Gibt die Tabelle einer Gruppe zu einer bestimmten Woche zurück

Anfrage

GET https://api.nitroleague.de/season/division-group/<ID>/standings/<WEEK>

        "name": string,
        "image": string,
        "identifier": string,
        "tag": string,
        "wonRounds": number,
        "lostRounds": number,
        "wonGames": number,
        "lostGames": number,

        "place": number
