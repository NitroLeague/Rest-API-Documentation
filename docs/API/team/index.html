team

/identifier​

GET

Gibt die Informationen über dieses Team mittels Identifier zurück

Anfrage

GET https://api.nitroleague.de/team/<Identifier>

Antwort

{
    "name": string,
    "tag": string,
    "identifier": string,
    "image": string
}

Beispiel

Anfrage

GET https://api.nitroleague.de/team/hngf95

Antwort

{
    "name": "Playing Ducks",
    "tag": "PD",
    "identifier": "hngf95",
    "image": "https://cdn.nitroleague.de/img/hngf95.png?bb7cafe7807b2c25001cd9716f5686a7"
}

/identifier/division​

GET

Gibt die Division, sowie Gruppe, in welcher sich dieses Team befindet zurück

Anfrage

GET https://api.nitroleague.de/team/<Identifier>/division

Antwort

{
    "id": number,
    "name": string,
    "groups": Array<{
            "name": string,
            "id": number,
            "divisionId": number
    }>
}

Beispiel

Anfrage

GET https://api.nitroleague.de/team/hngf95/division

Antwort

{
    "id": 18,
    "name": "Division 1",
    "groups": [
        {
            "name": "Division 1",
            "id": 81,
            "teams": [],
            "divisionId": 18
        }
    ]
}

[
    {
        "identifier": string,

GET
